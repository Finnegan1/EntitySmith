# Component Overview

## Tree

```
App
├── WorkspaceContext.Provider
│   └── DatasetContext.Provider
│       ├── KeyboardSave              — listens for Cmd/Ctrl+S, no DOM output
│       ├── WorkspaceSidebar          — 280 px fixed left panel
│       │   └── WorkspaceItem[]       — one collapsible group per workspace
│       │       └── WorkspaceFileItem[] — one button per file
│       ├── Separator (vertical)
│       └── MainContent
│           ├── EmptyState            — "Select a file" message
│           └── DatasetView           — shown when a file is selected
│               ├── DatasetError      — invalid files only
│               ├── DatasetHeader     — name, description, source, Save button
│               ├── DatasetTableToolbar
│               │   ├── AddAttributeDialog
│               │   └── RemoveAttributeDialog
│               └── DatasetTable
│                   └── DatasetTableCell[] — click-to-edit cells
└── Toaster                           — sonner toast container
```

## Layout

The root layout is a full-screen flex row:

```
┌─────────────────────┬─────────────────────────────────────┐
│  WorkspaceSidebar   │  DatasetView  (or EmptyState)        │
│  280 px fixed       │  flex-1, overflow-hidden             │
└─────────────────────┴─────────────────────────────────────┘
```

`DatasetView` is itself a flex column:

```
┌─────────────────────────────────────────────────────────┐
│  DatasetHeader          (name, description, Save btn)    │
├─────────────────────────────────────────────────────────┤
│  DatasetTableToolbar    (Add / Remove Attribute)         │
├─────────────────────────────────────────────────────────┤
│  DatasetTable           (scrollable, flex-1)             │
└─────────────────────────────────────────────────────────┘
```

## UI primitives

Everything in `components/ui/` is generated by **shadcn** and should not be edited manually. To add a new primitive:

```bash
# from apps/electron/
bunx shadcn add <component-name>
```

Current primitives: `alert`, `badge`, `button`, `dialog`, `dropdown-menu`, `input`, `scroll-area`, `separator`, `sheet`, `sidebar`, `skeleton`, `sonner`, `table`, `tooltip`.
